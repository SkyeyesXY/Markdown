##2.Python Web开发及Flask简介

###1)Flask简介
Flask是使用python编写的轻量级微内核Web应用框架(Django是全栈式),只有web开发的基础功能,但易于扩展,灵活

|-|Django|Flask|
|:---:|:---:|:---:|
|类型|全栈式|微内核|
|ORM|有|无|
|路由|有|有|
|模板|有|有|
|缓存|有|无|
|生成后台|有|无|
|数据验证|有|无|

##3.Flask开发环境搭建
###1)python包管理工具
> * easy_install
> * pip

**Python的包都是全局安装**

> * 优点:避免重复安装
> * 缺点:无法灵活使用不同版本

pip常用命令
> * pip install
> * pip uninstall
> * pip list

###2)windows下安装
> * 安装python
> * 安装pip
> * 安装flask:pip install flask
